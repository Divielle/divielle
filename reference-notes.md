# Filippo Ruffini Reference - Key Design Observations

## Overall Philosophy
- DARK background (near-black #0a0a0a) as default
- Extremely experimental, art-direction focused
- NOT a sales website — it's an EXPERIENCE
- Custom cursor (crosshair style, red/orange)
- Pixelated/glitch image reveal animations on scroll
- Numbered sections with parenthetical labels: ( 01 ), ( 02 ), ( 03 )
- Bracket/parenthesis motif throughout: ( ABOUT ), ( WORKS ), ( CONTACT )

## Layout Patterns
- Full-viewport hero with brand name as massive typography
- Corner-anchored metadata (top-left: "PERSONAL PORTFOLIO (2023)", top-right: "AVAILABLE FOR FREELANCE PROJECTS")
- Navigation as floating pill-style links with parentheses
- Asymmetric grid with lots of negative space
- Projects listed as large italic serif titles on left, with details appearing on hover/scroll
- Massive footer copyright year as oversized typography
- Thin border/outline boxes as decorative elements

## Typography
- Mix of mono/sans for labels and metadata
- Large serif italic for project/section titles
- Very small uppercase tracking for categories/labels
- Oversized display text for impact moments

## Color
- Primary: Dark background (#0a0a0a to #111)
- Accent: Orange/coral (#FF6B35 or similar)
- Text: White at various opacities
- Grey for secondary text
- Green for interactive elements (links)

## Interactions
- Pixelated image reveal (images appear as pixel blocks that resolve)
- Smooth scroll with sections snapping
- Hover states reveal project details/screenshots
- Custom cursor that changes on interactive elements
- Loading percentage counter on initial load
- "Scroll down" indicator at bottom

## Key Takeaways for DIVIELLE Adaptation
- Keep DIVIELLE gold (#D4AF37) as accent instead of orange
- Dark-first experience
- Numbered sections with parenthetical labels
- Massive typography moments
- Pixelated/glitch reveal for product images
- Custom cursor
- Corner-anchored metadata
- Experimental 3D as the centerpiece (not sales)
- Remove all "shop" and "buy" language — replace with experiential language
- Projects section becomes "Collections" with same hover-reveal pattern
- Footer with oversized year/copyright
